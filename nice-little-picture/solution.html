The description of the level says, that there is an image on the page. So let's open dev tools and check this.

In the document structure we see the invisible image encoded in base64 format.

Right click in the center of the page -> Save as anywhere you want.

So we have an image. In the properties we see that this image is in RGBA format. So some information can be hidden with help of Alpha-channel.

So let's grab any online image editor and apply color change.

So /two-shades-of-black is our result.
