So, let's follow the bits. First, we will open the dev tools f12. We see the following structure:

<!DOCTYPE html> <html lang="en"> <head> <link rel="stylesheet" href="styles.css"> <title>Code Quest - Level 1</title> </head> <body> <h1>Level 1</h1> <h2>Welcome stranger!</h2> <p>You made the first step in a challenging quest for enlightenment.</p> <p>Certainly, you're wondering what you're dealing with here and who set this up.</p> <p>With some perseverance and curiosity you'll find out soon enough.</p> <p>Follow the bits and be rewarded with unlimited fame and glory.</p> <!-- port++ --> </body> </html>

The "body" contains h1 and h2 heading, a couple of paragraphs and one comment.

The comment says "port++", so let's increment the port.
Default port for HTTP protocol is 80. So 80 + 1 = 81.
We can add a port to url using notaion "http://domain:port".

So our resul url will be: http://18.198.170.188:81/

If we go to this url we will see Level 2 page. ( task / solution )
